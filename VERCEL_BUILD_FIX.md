# Решение ошибки сборки на Vercel

## Проблема
Build Failed: Command "npm run build" exited with 1

## Возможные причины и решения:

### 1. Prisma Client не сгенерирован ✅ (Исправлено)

Добавлен `postinstall` скрипт и `prisma generate` в build команду.

### 2. Отсутствующие Environment Variables

Убедитесь, что в Vercel добавлены все необходимые переменные:

**Обязательные:**
- `DATABASE_URL` - строка подключения к PostgreSQL
- `AUTH_SECRET` - секретный ключ (можно сгенерировать: `openssl rand -base64 32`)

**Опциональные (для полной функциональности):**
- `NEXTAUTH_URL` - будет автоматически установлен Vercel
- `RESEND_API_KEY` - для отправки email
- `GOOGLE_CLIENT_ID` - для Google Calendar
- `GOOGLE_CLIENT_SECRET` - для Google Calendar

### 3. Проверьте логи сборки

В Vercel Dashboard:
1. Откройте последний деплой
2. Перейдите в **Build Logs**
3. Найдите точную ошибку (обычно в конце логов)

### 4. Частые ошибки:

**Prisma ошибки:**
- Решение: Убедитесь, что `DATABASE_URL` установлен
- Prisma Client теперь генерируется автоматически

**TypeScript ошибки:**
- Решение: Проверьте логи на конкретные ошибки типов

**Missing dependencies:**
- Решение: Убедитесь, что все зависимости в `package.json`

## После исправлений:

1. Закоммитьте изменения (уже сделано)
2. Vercel автоматически пересоберет проект
3. Или вручную: **Deployments** → **Redeploy**

## Если проблема осталась:

Пришлите полные логи сборки из Vercel (особенно последние строки с ошибкой).
